# AI-Project
Tic-Tac-Toe AI

A simple console-based Tic-Tac-Toe game in Python with two modes:

AI vs AI – Watch two computer players compete using AI algorithms.
Player vs AI – Play against an AI that uses the Minimax algorithm.
Features
Console-based gameplay
Two modes: AI vs AI & Player vs AI
Optimal AI using Minimax (cannot lose if played perfectly)
Step-by-step board updates in the terminal
How to Run

Clone the repo and run the desired script:

git clone https://github.com/mo7ammednabil/AI-Project.git
cd tic-tac-toe-ai

AI vs AI:

python tic_tac_toe_ai_vs_ai.py

Player vs AI:

python tic_tac_toe_player_vs_ai.py

Follow the on-screen instructions (for Player vs AI, enter your moves as row and column numbers 1-3).

Requirements
Python 3.7+
No external libraries
